# Ch Documentation

## About

The official website for [ch](../README.md), a lightweight AI CLI tool for terminal-based AI interactions. The site is hosted on **GitHub Pages** at https://mehmetmhy.github.io/ch/

## Usage

Run `./build.sh --help` to see all available commands, or `./build.sh --run` to open the website locally. To update website images, edit `logo.png` and run `./build.sh --convert`. This feature requires [ImageMagick](https://imagemagick.org/) to be installed.
